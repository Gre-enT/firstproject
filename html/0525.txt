신호 출석

입실, 매 시간, 퇴실


head : 링크, 연결 등
body : 내용

기간 : 2달
목표 : 간단한 반응형 사이트(게시판, 홈페이지 등) 직접 제작

카페 : https://cafe.naver.com/tiswebfront01

p태그 : 공백 1칸 이상 적용 X, 엔터 적용 X
pre태그 : 입력한 그대로 적용

ul ol dl li dt dd : 목록을 구성하는데 사용하는 태그



table 태그
tr : 행
td : 열
th : 열(굵음, 가운데 정렬)
colspan : 가로 합침
rowspan : 세로 합침

블록 레벨 : 부피가 있음(줄이 넘어감)
인라인 레벨 : 부피가 없음(줄이 넘어가지 않음, 글씨를 꾸며주는 태그, a태그 등)

예외 : img - 인라인 레벨이지만 부피가 있음(width, height가 존재함)

사용 프로그램 : vs코드
